# Security Policy

## Reporting a Vulnerability

Please report suspected Unforgit vulnerabilities privately by opening a GitHub security advisory in the main Unforgit repository:

https://github.com/MiguelMedeiros/unforgit/security/advisories/new

Do not include API keys, tokens, private keys, recovery phrases, or production `.env` values in issues, pull requests, logs, examples, or Unforgit memories.
