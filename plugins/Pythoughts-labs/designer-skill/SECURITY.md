## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues by emailing **mohamedmoradelkholy1@gmail.com** with the subject line `[designer-skill] Security Vulnerability`.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or proof-of-concept code
- Any suggested mitigations

You will receive an acknowledgement within 48 hours and a resolution timeline within 7 days. We will coordinate a fix and public disclosure with you.
