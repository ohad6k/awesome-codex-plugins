# Reality Check

Compare an explicit claim with observable evidence. Cite every confirmed or
missing behavior with a file, command result, or artifact. Separate:

- confirmed behavior;
- concrete gap;
- incomplete evidence;
- changed assumptions.

Return the report to the caller. Plan may use concrete gaps as input to a new
PlanPacket. Reality Check does not create work, schedule, claim, implement,
validate, retry, or deliver.
