# Механическая проверка изменения навыка

Этот шаблон не создаёт предметные правила и не переносит решения одного клиента
к другому. Он проверяет только целостность поставки.

## Вход

- перечень изменённых файлов;
- назначение изменения;
- команды испытаний;
- происхождение перенесённого материала;
- список удалённых или заменённых путей.

## Проверки

- [ ] каждый путь существует или явно удалён;
- [ ] все ссылки относительны и ведут к существующим файлам;
- [ ] новые сценарии разбираются интерпретатором и имеют испытания;
- [ ] шаблоны содержат только синтетические значения;
- [ ] нет ключей, токенов, клиентских названий, доменов и служебных номеров;
- [ ] сборщик не выносит смысловой вердикт;
- [ ] операция записи имеет точный пакет, разрешение, снимок, чтение результата
      и кандидата на возврат;
- [ ] установщик включает новый навык и не перезаписывает пользовательский файл;
- [ ] выпускная проверка завершилась без ошибок.

## Результат

Запишите только факты: проверенный путь, команда, код завершения и найденное
несоответствие. Выбор стратегии, минус-фразы, площадки, ставки, объявления и
другие экспертные решения остаются за отдельной смысловой проверкой.
