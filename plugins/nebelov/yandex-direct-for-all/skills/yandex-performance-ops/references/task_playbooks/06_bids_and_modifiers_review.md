# Ставки и корректировки: кандидаты

Расчёт сохраняет исходные показатели, формулу, ограничение и предложенное значение отдельно по каждой строке. Результат всегда имеет `candidate_status=pending_review` и пустое ручное решение.

Предложение не является командой записи. Применение допускается только для разрешённой схемой пары метода, после свежего снимка и отдельного подтверждения пакета.
