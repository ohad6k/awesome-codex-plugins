# Площадки РСЯ: ручная проверка кандидатов

1. Сохраните полный отчёт, период, модель атрибуции и контрольную сумму.
2. Формулы стоимости, конверсий и объёма дают только диагностические сигналы.
3. Каждая площадка остаётся отдельной строкой с исходными показателями и пустым решением.
4. Проверяющий оценивает достаточность данных, назначение площадки и риск потери охвата.
5. Только действующее ручное решение может войти в подтверждаемый пакет изменения кампании.

Автоматическое отключение площадок и перенос порогов между клиентами запрещены.
