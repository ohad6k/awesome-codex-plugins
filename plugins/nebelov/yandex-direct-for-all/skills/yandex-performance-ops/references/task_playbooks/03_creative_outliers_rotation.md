# Варианты ротации объявлений

Сценарий выявляет выбросы по показам, кликам и конверсиям, но возвращает только варианты-кандидаты. Для каждой строки сохраняются исходное объявление, период, показатели, причина сигнала, предложенный вариант и пустое ручное решение.

Остановка, архивация и добавление объявления оформляются разными подтверждёнными пакетами. До применения проверяются ограничения текста и снимок текущих объявлений; после — выполняется обязательное чтение.
