# Надзорная проверка

Заполнить перед чтением действующего кабинета, перед планом записи, перед записью и перед итогом.

| Проверка | Требуемое доказательство |
|---|---|
| Объём | Понятное имя клиента и перечень только его кампаний |
| Доступ | Свежий успешный запрос к нужной службе |
| Приложение | Токен принадлежит ожидаемому общему или явно настроенному приложению |
| Исходное состояние | Снимок затрагиваемых объектов и контрольные суммы |
| Семантика | Каждая исходная строка получила решение либо явный блокер |
| Разрешение | Явное разрешение на точный пакет записи |
| Ворота | Проверены срок, объём, блокировка, стоимость и возможность отката |
| Чтение после записи | Фактические поля совпали с утверждённым пакетом |
| Итог | Нет скрытого невыполненного объёма |

Если хотя бы одна обязательная строка не доказана, следующий этап не начинается.
