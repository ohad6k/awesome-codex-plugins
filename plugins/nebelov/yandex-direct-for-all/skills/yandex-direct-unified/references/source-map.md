# Карта происхождения правил

Единый навык заменяет старые точки входа yandex-performance-ops и yandex-direct-client-lifecycle. Узкие технические справочники сохраняются отдельно.

При изменении навыка:

1. Сопоставить новое правило с действующим основным порядком и правилами решений.
2. Проверить, что правило переносимо и не содержит клиента, доступа, локального пути или разового происшествия.
3. Обновить основной файл и только необходимую ссылку; не создавать параллельный канон.
4. Проверить все относительные ссылки и существование каждого упомянутого сценария.
5. Запустить проверки установки в чистых временных домашних каталогах Кодекса и Клода.
6. Выполнить поиск секретов и частных данных по итоговому дереву.

История частных проектов не является источником публичной настройки. Из неё переносятся только обобщённые правила, подтверждённые повторяемой проверкой.
