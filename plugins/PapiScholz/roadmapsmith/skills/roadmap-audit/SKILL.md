# RoadmapSmith Audit

Use this command when the user wants the post-sync mismatch summary after applying evidence-backed checklist updates.

## Required behavior

1. Run `roadmapsmith sync --audit --project-root .`.
2. Explain that this is the current sync-plus-audit flow, not a standalone read-only audit engine.
