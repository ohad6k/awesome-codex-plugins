# RoadmapSmith Zero

Use this command when the repository is empty or low-context and the user needs the discovery interview plus first roadmap generation.

## Required behavior

1. Run `roadmapsmith zero --project-root .`.
2. If the CLI is missing, explain the install path instead of improvising the workflow manually.
