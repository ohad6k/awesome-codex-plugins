# Security Policy

## Reporting Security Issues

Please report security issues privately by emailing hello@sarwagya.wtf or by opening a private security advisory on GitHub when available.

Do not disclose vulnerabilities publicly until they have been reviewed and a fix or mitigation is available.

## Scope

This repository contains a Codex plugin wrapper for Ontoly Agent Skills. It does not include hosted services, credential storage, or remote execution infrastructure.

Security issues in Ontoly itself should be reported to the main repository:

https://github.com/0xsarwagya/ontoly/security
