# Security Policy

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository or by contacting Gustavo Meilus through the email address listed on the GitHub profile for `gustavo-meilus`.

Include the affected version or commit, reproduction steps, expected impact, and any relevant logs or scanner output. Do not open a public issue for a vulnerability until a fix or mitigation is available.

## Scope

AIBoarding ships agent skills, documentation templates, and local hook scripts. Reports are in scope when they affect generated onboarding files, hook behavior, repository writes performed by the skills, or bundled templates.

## Expectations

I will acknowledge valid reports as soon as practical, coordinate a fix in a private branch when needed, and credit reporters on request.
