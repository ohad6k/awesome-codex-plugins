# Build Relationship Link Campaign Workflow

Use this workflow to plan campaigns where relationships create link opportunities.

## When to Use

- The user wants long-term link opportunities, not one-off cold outreach.
- Expert surveys, roundups, partner assets, or community resources are being planned.
- A niche has identifiable publishers, experts, partners, or advocates.

## Prerequisites

- Target market, asset idea, and business goal.
- Candidate people, publishers, partners, or communities.
- A tracking system for relationship notes.

**Reference**: `references/core/knowledge.md`, `references/core/rules.md`

## Workflow Steps

### Step 1: Map The Relationship Field

- [ ] List people, publishers, organizations, audiences, and current state.
- [ ] Capture recent work, topics, promotion behavior, and contact path.

### Step 2: Choose Value-First Actions

- [ ] Define useful actions for each segment.
- [ ] Confirm actions are genuine and visible enough to matter.
- [ ] Avoid asking for links at this stage unless the relationship already supports it.

### Step 3: Design Collaboration

- [ ] Pick format: expert survey, group interview, partner resource, webinar, local/community effort, or writing assignment.
- [ ] Write contribution prompts that are easy to answer and interesting to participants.
- [ ] Plan credit, links, approvals, and promotion assets.

### Step 4: Execute Warm Outreach

- [ ] Reference real context and the value-first action.
- [ ] Ask for the smallest useful contribution or next step.
- [ ] Track every response and promised contribution.

### Step 5: Publish And Promote

- [ ] Notify contributors with ready-to-share assets.
- [ ] Promote contributors where appropriate.
- [ ] Ask for mentions or links only when the finished asset genuinely serves their audience.

## Exit Criteria

Task is complete when:

- [ ] Relationship states and next actions are documented.
- [ ] The collaboration has clear value for participants and readers.
- [ ] Link asks are timed to trust and asset usefulness.
