# Linkable Content Architecture Knowledge

Core concepts for preparing linkable assets for promotion.

## Key Concepts

### Orphan Asset

A linkable asset is orphaned when crawlers or users cannot reach it through internal links. Outreach to orphaned URLs can waste link equity and reduce discovery.

### Content Hub

A hub connects related assets, supporting pages, and business pages. It helps users navigate and gives external links a clearer path into the site.

### Link Equity Flow

External links to useful assets should support adjacent educational and commercial paths through natural internal links, not forced conversion blocks.

### URL Stability

Campaign URLs should be stable, canonical, searchable, and redirect-clean before publishers are asked to link.

### Promotion Readiness

An asset is ready when it is crawlable, internally linked, useful on its own, and connected to sensible next steps.

## Helpful Tools

- Site crawlers for orphan pages, status codes, canonicals, and internal links.
- Backlink exports to map inbound link destinations.
- Analytics and Search Console for traffic, queries, and engagement.
- CMS or codebase access for links, navigation, redirects, and templates.
- Log files or crawl stats for high-risk technical audits.
- Visualization tools for internal link graphs.
