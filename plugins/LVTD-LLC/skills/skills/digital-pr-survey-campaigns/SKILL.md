# Digital PR Survey Campaigns

Use this skill to turn surveys, expert panels, or proprietary data into credible digital PR assets that journalists and publishers can cite.

This skill is derived from Garrett French and Eric Ward's *Ultimate Guide to Link Building* and uses transformed guidance with durable book-topic references. Do not copy book prose into user outputs.

## Quick Start

1. Use the Contents table below to choose the smallest useful reference set.
2. Use `workflows/design-survey-pr-campaign.md` for a full campaign plan.
3. Confirm the story is newsworthy before collecting data.
4. Design the sample, questions, and methodology so an editor can trust the result.
5. Package findings for both journalists and linkable evergreen citation.

## Default Output

When asked for survey-led PR help, return:

1. **Story frame** - headline angle, audience, why now, and likely publishers.
2. **Data plan** - sample, source, question design, method limits, and quality checks.
3. **Asset plan** - report page, charts, methodology, quotes, and citation targets.
4. **Outreach plan** - exclusives, embargoes, reporter segments, pitch angle, and follow-up.
5. **Tool choices** - survey/data collection, analysis, media research, outreach, and measurement tools.
6. **Risks** - weak sample, leading questions, thin story, compliance, or unsubstantiated claims.

## Core Posture

- Data earns links only when it is credible, surprising, and useful to a publisher's audience.
- The methodology is part of the asset, not an afterthought.
- Reporters need a story, not a spreadsheet.
- Evergreen report pages should remain useful after the PR spike.
- Do not inflate findings beyond what the sample and questions support.
