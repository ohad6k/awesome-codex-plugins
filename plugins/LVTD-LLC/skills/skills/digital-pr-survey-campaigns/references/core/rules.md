# Digital PR Survey Campaigns Rules

Use these rules when designing or reviewing survey-led PR campaigns.

## Core Rules

### 1. Start With The Story

- Write the likely headline before writing survey questions.
- Name the publisher audiences that would care.
- Reject concepts that are only interesting to the brand.

### 2. Make The Method Defensible

- State sample source, sample size, field dates, question wording, and exclusions.
- Avoid leading, double-barreled, or vague questions.
- Do not imply population-level certainty when the sample cannot support it.

### 3. Package For Citation

- Put methodology, key findings, charts, and source attribution on the report page.
- Provide concise summaries reporters can verify quickly.
- Use stable URLs and section anchors for major findings.

### 4. Segment Pitches By Beat

- Match each angle to a reporter's demonstrated coverage.
- Give high-value targets a sharper hook than the generic pitch.
- Track exclusives, embargoes, and prior contact history.

### 5. Measure Quality, Not Only Coverage Count

- Record publication relevance, link status, link attributes, referral traffic, and relationship value.
- Note unlinked coverage for reclamation.

## Red Flags

- Sample is too small or unclear for the claim.
- Questions appear designed to manufacture a desired statistic.
- The pitch hides brand involvement.
- Visuals are prettier than the underlying finding.
- Outreach begins before the report page is live and citable.
