# Directory Link Evaluation Rules

Use these rules when deciding whether to submit to, update, or reject a directory.

## Core Rules

### 1. Classify The Directory

- Identify whether it is local, niche, association, marketplace, resource, award, or generic SEO directory.
- Apply the right standard for that type.

### 2. Check Real Audience Fit

- Confirm category, geography, profession, or buyer intent fit.
- Look for evidence that users or industry participants would actually use it.

### 3. Inspect Editorial Quality

- Review submission standards, moderation, listing depth, update recency, and neighboring listings.
- Prefer directories that reject irrelevant submissions.

### 4. Evaluate Link Risk

- Check whether outbound links are natural, placed across the whole site, sponsored, qualified, or surrounded by spam.
- Treat paid inclusion as a risk review, not a simple SEO tactic.

### 5. Decide And Track

- Submit only when the listing is useful beyond link value.
- Track login, renewal, cost, category, destination URL, link attributes, and review date.

## Hard Rejects

- Generic directory with no real audience or category standards.
- Obvious link-selling footprint.
- Followed links across the whole site sold for ranking purposes.
- Irrelevant category or geography.
- Low-quality outbound neighbors or malware/adult/pharma clusters.
