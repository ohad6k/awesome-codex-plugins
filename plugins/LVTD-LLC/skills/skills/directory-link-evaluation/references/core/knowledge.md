# Directory Link Evaluation Knowledge

Core concepts for judging directory, listing, and citation opportunities.

## Key Concepts

### Directory Types

Common types include local citations, industry directories, professional associations, resource indexes, marketplaces, partner lists, award lists, and paid listings.

### User Utility

A directory is more defensible when real users rely on it to find, compare, verify, or contact relevant organizations.

### Editorial Standard

Quality directories use clear categories, review submissions, maintain listings, reject irrelevant entries, and avoid obvious paid-link footprints.

### Paid Inclusion Risk

Paid listings are not automatically bad, but links intended to influence rankings need extra caution and may require qualified link attributes such as sponsored or non-ranking attributes.

## Helpful Tools

- Browser inspection for page quality, category fit, and outbound neighbors.
- Search operators for indexation and brand/category visibility.
- Backlink tools for link profile and spam signals.
- Crawler or browser extensions for link attributes and status codes.
- Local SEO citation tools when geography matters.
- Tracking sheet for submissions, costs, credentials, and renewal dates.
