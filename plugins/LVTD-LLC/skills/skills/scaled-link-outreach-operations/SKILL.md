# Scaled Link Outreach Operations

Use this skill to run higher-volume link outreach without losing relevance, deliverability, tracking discipline, or compliance hygiene.

This skill is derived from Garrett French and Eric Ward's *Ultimate Guide to Link Building* and uses transformed guidance with durable book-topic references. Do not copy book prose into user outputs.

## Quick Start

1. Use the Contents table below to choose the smallest useful reference set.
2. Use `workflows/operate-scaled-outreach.md` for a full operating pass.
3. Scale only after the offer and prospect segment are proven.
4. Pilot a small batch before sending the whole list.
5. Diagnose replies by segment, offer, subject, personalization, and deliverability.

## Default Output

When asked for scaled outreach help, return:

1. **Campaign model** - offer, segment, list source, send volume, and risk constraints.
2. **Segmentation and variables** - what uses templates, personalization, suppression, and manual review.
3. **Pilot plan** - sample size, success thresholds, failure signals, and expansion rule.
4. **Operations checklist** - deliverability, dedupe, opt-out, follow-up, ownership, and tracking fields.
5. **Tool choices** - outreach CRM, spreadsheet/database, email verification, mailbox health, analytics, and compliance references.
6. **Diagnostics** - reply-rate issues, bounce causes, complaint risk, placement quality, and next experiment.

## Core Posture

- Scale relevance, not just send volume.
- Templates are acceptable only when the audience, offer, and context still fit.
- Pilot sends protect the domain, brand, and list quality.
- Compliance and opt-out handling are operating requirements.
- Reply rate alone is not success if placements are weak or risky.
