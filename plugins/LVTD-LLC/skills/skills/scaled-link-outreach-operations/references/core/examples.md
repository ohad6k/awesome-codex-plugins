# Scaled Link Outreach Operations Examples

Use these examples to design templates, pilots, and diagnostics.

## Segment Definition

| Field | Example |
|-------|---------|
| Segment | University career resource pages with outdated remote-work guide links |
| Shared context | Curators maintain career resources for students |
| Evidence variable | Page URL, dead URL, replacement asset, category |
| Ask | Suggest replacement resource |
| Manual review | Top-tier universities and pages with named editors |

## Pilot Thresholds

| Metric | Review Trigger |
|--------|----------------|
| Bounce rate | Data quality problem or mailbox risk |
| Spam complaints | Stop and review consent/source/wording |
| Reply rate | Diagnose subject, offer, relevance, and sender |
| Positive replies | Expand only if placements are relevant and safe |
| Negative replies | Revise segment or stop campaign |

## Template Review

```
Subject: Resource update for [page topic]

Hi [Name],

I noticed [page URL] still includes [specific outdated/dead resource]. We published [replacement resource] for [audience/task], including [proof point].

If you update that section, this may be a useful replacement: [URL]

Either way, thanks for maintaining the resource.
```

## Diagnostic Pattern

| Symptom | Likely Cause | Next Action |
|---------|--------------|-------------|
| Low opens | Subject, sender trust, deliverability | Test subject and mailbox health |
| Opens but no replies | Offer or relevance weak | Re-segment and strengthen evidence |
| Replies but few links | Asset lacks citation value | Fix asset or ask type |
| Many complaints | List or consent problem | Stop and audit compliance |
