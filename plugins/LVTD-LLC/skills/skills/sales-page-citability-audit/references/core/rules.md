# Sales Page Citability Audit Rules

Use these rules when reviewing commercial pages for link readiness.

## Core Rules

### 1. Name The Citation Scenario

- Identify who would cite the page and in what kind of content.
- State the reader task the link supports.
- Reject citation scenarios that exist only for keyword or link value.

### 2. Separate Conversion Copy From Reference Value

- Keep the primary offer clear.
- Add citable modules that help users evaluate, understand, or act.
- Do not bury useful material behind a form if outreach needs open citation.

### 3. Build Specific Link Targets

- Add stable section anchors for definitions, data, tools, examples, FAQs, and templates.
- Use internal links from supporting content and hubs.
- Avoid asking editors to link to a generic sales page when a better support page is needed.

### 4. Support Claims With Evidence

- Add methodology, examples, sources, customer evidence, expert credentials, or transparent criteria.
- Remove unsupported superlatives and vague claims from citation targets.

### 5. Preserve User Experience

- Do not overload the page with unrelated content.
- Keep page speed, mobile layout, and conversion path intact.

## Red Flags

- The only link reason is "this is our product."
- The page has no section anchors or source-like elements.
- Claims are promotional but unsupported.
- Outreach wants editorial links before page changes are made.
- The page is blocked, canonicalized elsewhere, or hard to crawl.
