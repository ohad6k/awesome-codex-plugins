# Sales Page Citability Audit Knowledge

Core concepts for making commercial pages easier to cite in editorial contexts.

## Key Concepts

### Citability

A page is citable when a publisher can link to it because readers need the page's definition, data, tool, example, comparison, template, expert explanation, or proof.

### Commercial Page Gap

Sales pages often explain why to buy but not why an editor should cite them. Add reference value without hiding the commercial purpose.

### Practitioner Action Cycle

Map what the audience does before, during, and after the buying or problem-solving moment. Citable modules should support that real workflow.

### Section-Specific Modules

Specific sections make commercial pages easier to cite: data points, definitions, calculators, templates, process diagrams, examples, FAQs, expert notes, and transparent comparison criteria.

## Helpful Tools

- Site crawler for page status, internal links, and anchors.
- Analytics and Search Console exports for audience questions and page performance.
- CRM, support, sales-call, and review data for real objections and tasks.
- CMS/page builder for section anchors and content modules.
- Structured data testing when adding FAQ, product, review, or how-to markup.
