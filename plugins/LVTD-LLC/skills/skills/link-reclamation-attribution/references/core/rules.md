# Link Reclamation Attribution Rules

Use these rules when evaluating missed-link and attribution opportunities.

## Core Rules

### 1. Verify The Existing Reference

- Confirm the page references the brand, asset, person, data, image, or old URL.
- Capture the exact context and screenshot when useful.
- Check whether the current link is missing, broken, qualified, redirected, or pointed incorrectly.

### 2. Fix Owned Infrastructure First

- Redirect valuable dead owned URLs.
- Repair canonical and redirect issues before contacting publishers.
- Avoid asking publishers to update links that your site cannot resolve cleanly.

### 3. Ask For Attribution, Not SEO Value

- Explain the factual correction or reader benefit.
- Include exact source and destination URLs.
- Keep the request short and non-accusatory.

### 4. Prioritize By Value And Legitimacy

- Highest priority: trusted publisher, clear attribution need, relevant page, easy contact path.
- Lower priority: weak relevance, old content, no editor path, or minor mention.
- Reject: scraper spam, toxic sites, irrelevant pages, or negative contexts where a link would amplify harm.

### 5. Verify The Outcome

- Check live link destination, anchor, link attributes, and page status.
- Record no-response and no-action decisions.

## Red Flags

- The mention is too vague to justify a link.
- The publisher used the brand in criticism or controversy.
- The page is scraped, spun, or ad-saturated.
- The request sounds like a copyright threat when the user asked for normal outreach.
- The destination URL is unstable or irrelevant.
