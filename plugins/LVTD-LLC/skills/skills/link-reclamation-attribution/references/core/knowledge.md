# Link Reclamation Attribution Knowledge

Core concepts for recovering links from existing mentions, citations, and historical URL evidence.

## Key Concepts

### Reclamation Opportunity

A reclamation opportunity exists when a publisher already references the brand, asset, image, data, product, person, or URL but the link is missing, broken, outdated, or pointed at the wrong destination.

### Attribution Fit

The strongest requests are factual corrections: "this image/report/tool came from here" or "this old URL now lives here." Weak requests ask for a link where no reader benefit or attribution need exists.

### Owned URL Salvage

Broken backlinks to owned pages often deserve redirects before external outreach. Fix what the site controls first.

### Mention Tone

Positive and neutral mentions may support a polite attribution ask. Negative mentions need care; sometimes a correction, response, or no action is better than asking for a link.

## Helpful Tools

- Search operators and mention monitoring for unlinked brand or asset mentions.
- Reverse image search for image attribution.
- Backlink crawlers for lost links and broken owned URLs.
- Site crawlers for redirect chains and destination checks.
- Outreach CRM or spreadsheet for contacts, requests, and verification.
