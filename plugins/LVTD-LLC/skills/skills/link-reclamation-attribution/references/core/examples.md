# Link Reclamation Attribution Examples

Use these examples to structure audits and requests.

## Opportunity Scoring

| Signal | Strong | Weak |
|--------|--------|------|
| Reference | Uses your data, image, quote, or old URL | Generic brand name in passing |
| Context | Relevant editorial page | Scraper, forum spam, or unrelated page |
| Fix | Link helps readers find source | Link only helps SEO |
| Effort | Clear editor or contact path | No contact path or abandoned site |

## Attribution Request

```
Subject: Source link for [asset/stat/image] on [page topic]

Hi [Name],

Thanks for referencing our [report/image/tool] in your article on [topic]. The source URL is [destination URL] if you want to give readers the original methodology/details.

The mention is here: [publisher URL]

Thanks,
[Name]
```

## Broken Owned URL Salvage

| Finding | Action |
|---------|--------|
| Old report URL has high-quality backlinks | 301 redirect to the current report or closest equivalent |
| Old URL has no replacement | Consider rebuilding the resource before outreach |
| External page links through a redirect chain | Fix redirect chain first, then optionally request direct update |

## No-Action Cases

- Negative mention where a link would increase visibility.
- Scraped copy that offers no trust or traffic.
- Publisher already cites a better source.
- The only possible destination is a sales page with no source value.
